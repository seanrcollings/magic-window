# Magic Window
`magic-window` is a CLI tool to fetch and dismiss a window / container for i3 and sway.

# Usage
Make the currently focused container the magic window:
```
$ magic-window pick
```
Hide it with:
```
$ magic-window hide
```
Show it with:
```
$ magic-window show
```
Toggle between show / hide
```
$ magic-window toggle
```
